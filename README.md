# 🚀 Artemis Is Toilet GO

A minimal Chrome Extension that answers one critical question:

> **Is the Artemis toilet GO or NO-GO?**

---

## Overview

This project is a lightweight Chrome Extension

Its purpose is simple:

- Determine a binary status: **GO / NO-GO**
- Expose that status on the browser navbar

---

## Installation (Development Mode)

1. Open Chrome
2. Go to: `chrome://extensions/`
3. Enable **Developer mode** (top right)
4. Click **Load unpacked**
5. Select this project folder
